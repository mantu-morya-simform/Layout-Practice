# Layout Practice Project

This project contains a collection of HTML and CSS layout exercises designed to practice various layout techniques using CSS Grid, Flexbox, and other positioning methods.

## Project Structure

### Main Folders (Days 1-5)

- **One/**: Basic layout with header, main content, and sidebars using(Flex and Grid)
- **Two/**: Card-based layout with left and right sections using(Flex and Positioning methods)
- **Three/**: Grid-based layout with multiple HTML files using(Flex and Grid)
  - `one.html`: CSS Grid layout with header, navigation, aside, and content areas
  - `two.html`: Vatshal Review-1 layout
- **Four/**: Satyam Review-1 layout
- **Five/**: Krish Review-1 layout

### Weekend Practice

- **weekend-1/**: practice dashboard-style layouts
  - **One/**: Admin dashboard layout with sidebar navigation using(Flex and Grid)
  - **Two/**: Additional dashboard variations with header, sidebar, navigation section using(Flex and Grid)
  - **Three/**: complex layout exercises using(Flex and Grid)

Each folder contains:

- `index.html` (or `one.html`, `two.html` in Three/): The HTML structure
- `index.css`: The CSS styles for the layout
- Also Each Layout Its Output Screenshot are there

## Learning Objectives

This project helps practice:

- CSS Grid layouts
- Flexbox layouts
- Semantic HTML structure
- CSS positioning techniques
